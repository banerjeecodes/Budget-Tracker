# HTML Based offline Budget Appication
This is a simple HTML Application that you can run offline to analyse your budget.
  1. The app can save your analysed transactions as a JSON File for future reference.
  2. New transaction statements can be loaded with a csv file.

## FORMAT:
|date |	party	| withdrawal | deposit | balance |
|-----|-------|------------|---------|---------|
|DD-MM-YYYY|NAME OF THE PARTY| WITHDRAW AMOUNT | DEPOSIT AMOUNT | BALANCE AMOUNT|

## Features:
- Categorise all your transactions into buckets
- Graph view to show monthly breakup of each category
- Monthly breakup of overall expenses into different categories
- Drilldown to show individual transactions of each month
- Easy CSV Load to add new transactions
- Download analysed file into JSON for future use
- Re-categorise your expenses in-app.
